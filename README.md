# Gato Interactivo

Esta es una aplicación interactiva de un gato dibujado con Python y Pygame.

## Requisitos

- Python 3
- Pygame

## Instalación

1. Crea un entorno virtual (opcional pero recomendado):

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # En Linux/Mac
    # .venv\Scripts\activate  # En Windows
    ```

2. Instala las dependencias:

    ```bash
    pip install -r requirements.txt
    ```

## Ejecución

Para correr la aplicación:

```bash
python main.py
```

## Controles

- **Cualquier tecla**: Hace que el gato haga algo diferente (saltar, maullar, cambiar fondo, perseguir pelota, saludar, etc.).
- **Salir**: Presiona `Ctrl` + `Alt` + `E` + `X` simultáneamente.
